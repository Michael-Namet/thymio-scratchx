Introduction

This Scratch extension let you program the educational robot Thymio using the scratch language. You can control LEDs, let Thymio wonder around avoiding obstacles using its proximity sensors and the 2 independent motors, follow black line on the ground or read barcodes, draw shapes putting a felt pen in Thymioâ€™s pen holder, play and record sounds, interact with the 3 axes accelerometer or measure the temperature.

You need to have the AsebaHttp bridge installed (see Getting Started section) and running on your PC or laptop and a Thymio Robot switched on to program it using the ScratchX extension.

What do you need

1) Thymio Robot wireless or wired to your PC using the USB cable
2) USB Cable to charge Thymio and to program it if you do not have the wireless dongle version
3) A laptop or PC to program using ScartchX

Ready to start

Blocks translations have been provided in the following languages

en - English
fr - French

To set the block language, add &lang=xx to the ScratchX URL where xx is one the 2 language codes above.

Example for French:
http://scratchx.org?url=http://thymio.github.io/scratch-thymio-extension/thymio_extension.js&lang=fr

If you would like to help out by providing block translations for another language, please see the Github issue.
